About

Saya Candra Yusuf Irawan jurusan sistem Komputer Universitas Borobudur Lulusan tahun 2022. Aktivitas saya saat ini adalah sebagai jobseeker/pencari kerja dan berlatih meningkatkan kemampuan tentang penggunaan bahasa pemrograman JavaScript dan Git version control system. Saya memiliki minat karir sebagai Frontend Developer dan mampu dalam melakukan pembuatan website menggunakan HTML, CSS, JavaScript dan framework Bootstrap. Saya juga pernah aktif dalam kegiatan Organisasi pada acara LDKOM (Latihan dasar kepemimpinan Organisasi Mahasiswa) 2019 di Bogor, pada acara ini saya ditugaskan sebagai Pengaman yang menjaga keamanan seperti keributan dan keonaran yang dilakukan oleh para Mahasiswa baru.

Kampus saya mengajarkan saya tentang basic Programming, Teliti terhadap kesalahan variabel, teliti terhadap titik koma, mengajarkan bagaimana cara bertanggungjawab, berkomunikasi dengan baik dalam tim, kerjasama tim, mengatasi masalah, dan mengajarkan bagaimana caranya bekerja keras.
