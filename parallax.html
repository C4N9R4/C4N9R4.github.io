Pengertian Teks Editor

Text editor merupakan sofware perangkat lunak yg digunakan para programmer untuk menulis, menyusun kode, dan membaca kode program. Secara umum, text editor menyajikan beragam fitur, mulai yang sederhana sampai yang lengkap dengan berbagai tambahan plugin yang tersedia.

Emmet adalah salah satu fitur yang membantu mempermudah penulisan sintaks di editor program. tidak memakan banyak waktu dalam menulis semua tag, seperti tanda kutip, tanda kurung dll. Sebagai contoh jika teman-teman kalian ingin membuat semua tag HTML, hanya dengan tekan shift lalu tanda seru, kemudian tab, maka setelah itu akan tampil seluruh tag-tag HTML.

Visual Studio Code (VS Code) adalah sebuah teks editor multiplatform yang komplit dan handal buatan Microsoft.

  2. Ketik p lalu tekan tab
